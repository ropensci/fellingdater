# Overview of available sapwood data sets in the `fellingdater` package

This function provides an overview of the available sapwood data sets
and models within the `fellingdater` package.

## Usage

``` r
sw_data_overview()
```

## Value

A `character` vector with the names of all available data sets with
sapwood counts in the `fellingdater` package.

## Examples

``` r
# Get an overview of available sapwood data sets
sw_data_overview()
#>  [1] "Brathen_1982"     "Hollstein_1980"   "Miles_1997_NM"    "Miles_1997_SC"   
#>  [5] "Miles_1997_WBC"   "Pilcher_1987"     "Sohar_2012_ELL_c" "Sohar_2012_ELL_t"
#>  [9] "Sohar_2012_FWE_c" "Sohar_2012_FWE_t" "Wazny_1990"       "Weitz_2025"      
#> [13] "vanDaalen_NLBE"   "vanDaalen_Norway"
```
